# Backend con Express y GitFlow

## Objetivo
Configurar un entorno backend con **Express**, aplicar **GitFlow** con GitKraken, levantar un servidor básico con un endpoint raíz y documentar el proyecto.

---

## Requerimientos
- Node.js v18+ y npm v9+.  
- Git y GitKraken.  
- VS Code.  
- Cuenta en GitHub.  
- Conexión a Internet.
